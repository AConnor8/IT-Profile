My Project

My project is to develop a cloud storage/program where users are able to upload photo’s/videos from their devices onto the cloud platform for ‘First line’ of storage.

Next is to develop a program that continuously scans the storage space, makes copies and sends them to a dedicated NAS/Secondary storage system for better navigating/interaction.

The program will be able to copy and send to the second line of storage in an ‘organised’ format where users are able to edit the folders where photo’s/videos are placed for better interaction/navigating.

There would also be a ‘Third’ backup option in its ‘raw’ format where files/folders are directly copied from the first line of storage into the third line backup storage

My motivation for this project is to have a similar setup at home, where i(or my family) are able to upload files through a cloud storage platform anywhere in the world and always have a backup of it at home. As my family are massive amateur photographers who capture anything that moves, and every meal eaten.

Project Features

The product itself is mainly the Cloud platform/program that needs to store an abundance of high res photo/videos. The user will have an option to either save in its ‘original’ format or save in lower resolution and the ‘original’ format will be sent to the third storage/master back up system.

Users will be able to name/date and attach a ‘Tag’ to allow users to search specific photos that have the tagged word. The ability to assign to people/places using the ‘tag’ feature. If they want to search for a particular photo with ‘brother’ at location ‘home’ they would be able to just by adding those tagged words and using the search function.

Users will be able to have the assurance that their files/photos/videos will always have a secondary/master backup when uploading onto the cloud platform. They will also be able to ‘revert’ to previous back up file where needed.

The system will be able to continuously scan for changes to files/photos and send original copies to both the NAS and master back up. It will also be able to provide a detailed log of what changes has been to each file/s and provides an image to confirm successful upload onto the backup server.

Admin users or master account holders will be able to collage images into a temporary folder and send to friends/family for viewing. i.e. if an admin account user wants to provide pictures/videos of her 30th birthday, she would be able to use the search function and type ‘Admin 30th Birthday’, choose which they would like them to view and it will create a temporary folder which can be accessed through a unique link provided by the admin user. They can increase security by having it timed lock/expire, add a password, or just be able for view for anyone/everyone. They can also add the ability to restrict viewers to not be able to copy/save/download the shared files.

Users are not restricted by the use of using the dedicated app on their device, they can also use our website, where they can directly login and upload where needed.
